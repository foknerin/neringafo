Game Rock Paper Scissors

I've created a game Rock Paper Scissors (RPS). Sketched icons of this game and applyed responsiveness.

Game rules: RPS is a zero-sum hand game usually played between two people, in which each player simultaneously forms one of three shapes with an outstretched hand. These shapes are rock, paper and scissors. The game has only three possible outcomes other than a tie: rock crushes scissors; paper covers rock; scissors cut paper. If both players choose the same shape, the game is tied and is usually immediately replayed to break the tie. The game is often used as a choosing method in a way similar to coin flipping or throwing dice.
